# React: Componentes e JSX
## Aprendendo a construir componentes reutilizáveis 🚀

### 💻 Projeto: CodeConnect
Rede social para pessoas desenvolvedoras, desenvolvido durante o curso React: componentes e JSX da Alura.

---

### ✅ Funcionalidades do projeto
- Página de login responsiva, adaptada para desktop e celulares
- Formulário com e-mail, senha, opção "Lembrar-me" e recuperação de senha
- Acesso rápido com ícones do GitHub e LinkedIn
- Link direto para criação de novo cadastro
- Layout moderno e organizado, seguindo o design do Figma

---

### 🛠️ Técnicas e tecnologias utilizadas
- React: Biblioteca JavaScript para construção de interfaces
- JSX: Sintaxe que combina HTML e JavaScript, usada em todo o projeto
- Componentização: Criação de elementos reutilizáveis (Botão, CampoDeDigitação, Checkbox, Título)
- Props: Passagem de dados entre componentes
- Estilização com CSS: Layout flexível e responsivo com Media Queries
- Vite: Ferramenta para desenvolvimento rápido e build do projeto
- Git/GitHub: Controle de versão e compartilhamento do código
- Vercel: Deploy automático e hospedagem do site
